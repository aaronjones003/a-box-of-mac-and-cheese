# A Box of Mac and Cheese

> Web app that generates "A (Blank) of (Blank) and (Blank)" book covers

## Overview

A fun web application that generates random book covers in the style of "A Box of Mac and Cheese". Click the generate button to create humorous and random book title combinations.

## Features

- Random book title generation
- Visual book cover creation
- Bootstrap-styled interface
- Simple one-click generation

## Tech Stack

- HTML5
- Bootstrap 5.3.3
- Vanilla JavaScript
- Canvas API for image generation

## Live Demo

https://onigiri.zone/a-box-of-mac-and-cheese/

## Development

Open `index.html` in a web browser to run locally. No build process required.
